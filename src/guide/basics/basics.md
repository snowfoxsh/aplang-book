# Basics

Contrats you now have successfully installed ApLang and configured your development envierment! If you havent done that yet please see [Installation](installation.md). 

Intruducing the `.ap` file. For leagal reasons it doent stand Advanced Placment it stands for something else.... uhhh... Apple Pie! Yep! This is where you place your code. When you are ready to run your code then call in the interpreter that you just installed. Navigate to where your source code file and run in your console.
```bash
aplang yourfile.ap
```
`aplang` will then execute your file. Watch your terminal for any output your code may produce.
